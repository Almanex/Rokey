# Roke - Keyboard Layout Converter

Конвертирует выделенный текст из одной раскладки клавиатуры в другую (EN ↔ RU).

## Использование

1. Выделите текст в любом приложении
2. Нажмите **Ctrl+Shift+Q** (или **Ctrl+Alt+Q**, или **F12**)
3. Текст автоматически сконвертируется в правильную раскладку

## Примеры

- `ghbdtn` → `привет`
- `привет` → `ghbdtn`
- `rhjcnf` → `работа`

## Сборка

```bash
cmake -B build -G "Ninja"
cmake --build build
```

Исполняемый файл: `build/bin/Roke.exe`

## Работает в

- Блокнот
- Браузеры (Chrome, Firefox, Edge)
- IDE (VS Code, и др.)
- Все приложения с текстовыми полями
